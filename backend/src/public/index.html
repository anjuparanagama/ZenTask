ZenTask API

API Status: Online

API Endpoints

Clean, RESTful routes for authentication & task operations.

Authentication

POST /api/auth/register

Create a new user account and return a JWT token.

  • body name string
  • body email string
  • body password string
POST /api/auth/login

Authenticate user and return JWT token.

  • body email string
  • body password string

Tasks

All task routes require the header: Authorization: Bearer <token>

GET /api/tasks

Retrieve all tasks for the authenticated user.

  • header Authorization Bearer token
GET /api/tasks/:id

Fetch a single task by its unique identifier.

  • header Authorization Bearer token
  • param id number
POST /api/tasks

Create a new task for the authenticated user.

  • header Authorization Bearer token
  • body title string
  • body description string
  • body priority string
  • body status string
  • body due_date date
PUT /api/tasks/:id

Update an existing task by its unique identifier.

  • header Authorization Bearer token
  • param id number
  • body title string
  • body description string
  • body priority string
  • body status string
  • body due_date date
DELETE /api/tasks/:id

Delete a task by its unique identifier.

  • header Authorization Bearer token
  • param id number

Dashboard

All dashboard routes require the header: Authorization: Bearer <token>

GET /api/dashboard/all-tasks-count

Get task counts grouped by status for the authenticated user.

  • header Authorization Bearer token
GET /api/dashboard/overdue-tasks-count

Get the count of overdue tasks for the authenticated user.

  • header Authorization Bearer token
